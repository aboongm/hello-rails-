# Hello Rails App

Hello Rails Project implements a rails project that show "Hello Rails!" 
on the landing page which is localhost:3000



## Built With

- Ruby
- Rails
- PostgreSQL

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- A source-code editor (preferably Visual Studio Code)

### Install

- Ruby
- Rails
- PostgreSQL

### Using it Locally

- Clone the project

git clone https://github.com/aboongm/hello-rails-.git

cd hello-rails-


- To Start the programs

open terminal
run the command $irb (interactive ruby shell)
run the program functions
